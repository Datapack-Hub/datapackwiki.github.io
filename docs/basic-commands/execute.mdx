# Execute

The execute command will run a specified command relative to variables you input.

```mcfunction
# Basic Syntax
/execute <subcommand>
```
## Subcommands

### align
<table>
  <tbody>
    <tr>
      <td width="30%"><code> align &lt;xyz&gt; </code></td>
      <td> It aligns commands position to the corner of the current block. You can use x, y, and z in any combination and amount but you have to use at least one of them.</td>
    </tr>
  </tbody>
</table>

### anchored
<table>
  <tbody>
    <tr>
      <td width="50%"><code> anchored eyes </code></td>
      <td> Anchors the commands position to the executing entities eyes.</td>
    </tr>
    <tr>
      <td width="50%"><code> anchhored feet </code></td>
      <td> Anchors the commands position to the executing entities feet.</td>
    </tr>
  </tbody>
</table>

### as
<table sty>
  <tbody>
    <tr>
      <td width="50%"><code> as &lt;selector&gt; </code></td>
      <td> Executes the command as every entity that matches the selector but it does not move the commands position. </td>
    </tr>
  </tbody>
</table>

### at
<table sty>
  <tbody>
    <tr>
      <td width="50%"><code> at &lt;selector&gt; </code></td>
      <td> Executes the command at every entity that matches the selector but it does not executes as that entity. </td>
    </tr>
  </tbody>
</table>

### facing
<table>
  <tbody>
    <tr>
      <td>
        <table>
          <tr>
            <td width="50%"><code> facing &lt;x y z&gt; </code></td>
            <td> Executes the command facing the specified coordinates.</td>
          </tr>
        </table>
      </td>
    </tr>
    <tr>
      <td>
        <table>
          <tr>
            <td width="50%"><code> facing entity &lt;selector&gt; </code></td>
            <td> Executes the command facing an entity that matches the selector.</td>
          </tr>
        </table>
      </td>
    </tr>
  </tbody>
</table>
